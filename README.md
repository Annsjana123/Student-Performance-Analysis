# Student Performance Analysis

## Project Overview
This project analyzes student performance data to identify academic trends, performance factors, and at-risk students.

## Objectives
- Explore and clean student data
- Analyze performance trends
- Identify at-risk students
- Generate actionable recommendations
- Create visualizations for decision-making

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Performance Segmentation
4. At-Risk Student Identification
5. Visualization and Reporting

## Visualizations
- Score Distribution
- Attendance Analysis
- Study Hours vs Scores
- Performance Categories
- Correlation Heatmap
- At-Risk Student Analysis

## Key Findings
- Attendance strongly influences academic performance.
- Study hours positively affect student scores.
- A small group of students requires immediate academic support.
- Early intervention can improve student outcomes.

## Recommendation
Implement an Early At-Risk Student Identification System to proactively support struggling students.
